SELECT Command

INTRO

                  Used to fetch specific columns from a table.

SQL

                  SELECT name, marks FROM students;

Explanation

                  Displays only the name and marks columns from the students table.

OUTPUT

                  +----------+--------+
                  | name     | marks  |
                  +----------+--------+
                  | Neo      | 90     |
                  | Trinity  | 85     |
                  | Morpheus | 88     |
                  +----------+--------+

SELECT * Command

INTRO

                  Fetches all columns from the table.

SQL

                  SELECT * FROM students;

Explanation

                  Shows every column and row present in the students table.

OUTPUT

                  +----+----------+-------------+--------+
                  | id | name     | dept        | marks  |
                  +----+----------+-------------+--------+
                  | 1  | Neo      | CS          | 90     |
                  | 2  | Trinity  | IT          | 85     |
                  | 3  | Morpheus | CS          | 88     |
                  +----+----------+-------------+--------+

WHERE Clause

INTRO

                  Used to filter rows based on a condition.

SQL

                  SELECT * FROM students WHERE marks > 85;

Explanation

                  Only shows students who scored above 85.

OUTPUT

                  +----+----------+--------+--------+
                  | id | name     | dept   | marks  |
                  +----+----------+--------+--------+
                  | 1  | Neo      | CS     | 90     |
                  | 3  | Morpheus | CS     | 88     |
                  +----+----------+--------+--------+

GROUP BY CLAUSE

INTRO

                  Groups rows by one or more columns and aggregates data.

SQL

                  SELECT dept, AVG(marks) FROM students GROUP BY dept;

Explanation

                  Shows average marks grouped by department.

OUTPUT

                  +----------+-------------+
                  | dept     | AVG(marks)  |
                  +----------+-------------+
                  | CS       | 89.0        |
                  | IT       | 85.0        |
                  +----------+-------------+

HAVING CLAUSE

INTRO

                  Used to filter groups created by GROUP BY.

SQL

                  SELECT dept, AVG(marks) 
                  FROM students 
                  GROUP BY dept 
                  HAVING AVG(marks) > 85;

Explanation

                  Only departments with an average above 85 will be shown.

OUTPUT

                  +--------+-------------+
                  | dept   | AVG(marks)  |
                  +--------+-------------+
                  | CS     | 89.0        |
                  +--------+-------------+

SELECT DISTINCT CLAUSE

INTRO

SQL

Explanation

                  Shows only unique department names.

OUTPUT

                  +--------+
                  | dept   |

                  | IT     |

LIKE OPERATOR

INTRO

                  Used to search for a pattern in a column.

SQL

                  SELECT * FROM students WHERE name LIKE 'T%';

Explanation

                  Finds names starting with 'T', like 'Trinity'.

OUTPUT

                  +----+---------+--------+--------+
                  | id | name    | dept   | marks  |
                  +----+---------+--------+--------+
                  | 2  | Trinity | IT     | 85     |
                  +----+---------+--------+--------+

IN & BETWEEN OPERATORS

INTRO

                  IN checks if a value matches any value in a list.
                  BETWEEN checks for values within a range (inclusive).

SQL

                  SELECT * FROM students WHERE dept IN ('CS', 'IT');
                  
                  SELECT * FROM students WHERE marks BETWEEN 85 AND 90;

Explanation

                  IN filters by multiple matching values.
                  BETWEEN gives results within the given range.

OUTPUT

                  -- For IN:
                  +----+----------+--------+--------+
                  | id | name     | dept   | marks  |
                  +----+----------+--------+--------+
                  | 1  | Neo      | CS     | 90     |
                  | 2  | Trinity  | IT     | 85     |
                  | 3  | Morpheus | CS     | 88     |
                  +----+----------+--------+--------+
                  
                  -- For BETWEEN:
                  +----+----------+--------+--------+
                  | id | name     | dept   | marks  |
                  +----+----------+--------+--------+
                  | 2  | Trinity  | IT     | 85     |
                  | 3  | Morpheus | CS     | 88     |
                  +----+----------+--------+--------+

ANY & ALL OPERATORS

INTRO

                  ANY returns true if at least one subquery result matches.
                  ALL returns true if all values in the subquery match.

SQL

                  SELECT * FROM students WHERE marks > ANY (SELECT marks FROM students WHERE dept = 'IT');
                  
                  SELECT * FROM students WHERE marks > ALL (SELECT marks FROM students WHERE dept = 'IT');

Explanation

                  ANY checks if condition is true for any value.
                  ALL checks if condition is true for all values.

OUTPUT

                  -- ANY:
                  Shows students with marks greater than at least one IT student's mark.
                  
                  -- ALL:
                  Shows students with marks greater than all IT students' marks.

ALIAS (AS)

INTRO

                  Alias is a temporary name given to a table or column.

SQL

                  SELECT name AS student_name, marks AS score FROM students;

Explanation

                  AS renames columns for display. Doesn’t affect original table.

OUTPUT

                  +--------------+-------+
                  | student_name | score |
                  +--------------+-------+
                  | Neo          | 90    |
                  | Trinity      | 85    |
                  | Morpheus     | 88    |
                  +--------------+-------+
