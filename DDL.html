CREATE TABLE

INTRO

The CREATE TABLE command is used to define a structure within a database.
                      
                  Syntax:
                  CREATE TABLE table_name (
                    column1 datatype,
                    column2 datatype,
                    ...
                    );

SQL

                    CREATE TABLE students (
                      id INT PRIMARY KEY,
                      name VARCHAR(50),
                      age INT
                      );

Verify Creation

                      SHOW TABLES;

OUTPUT

          Query OK, 0 rows affected (0.03 sec)
              
          +------------------+
          | Tables_in_college|
          +------------------+
          | students         |
          +------------------+

CREATE TABLE (FROM ANOTHER TABLE)

INTRO

The CREATE TABLE AS SELECT command helps you create a new table from an existing table.
    
Syntax (with data):
CREATE TABLE new_table AS
SELECT * FROM existing_table;
    
Syntax (structure only):
CREATE TABLE new_table AS
SELECT * FROM existing_table WHERE 1 = 0;

SQL

Create new table with data from existing
CREATE TABLE backup_students AS
SELECT * FROM students;
    
Create new table with just structure
CREATE TABLE empty_students AS
SELECT * FROM students WHERE 1 = 0;

Verify Creation

SHOW TABLES;

OUTPUT

Query OK, 2 rows affected (0.02 sec)
    
+------------------+
| Tables_in_college|
+------------------+
| students         |
| backup_students  |
| empty_students   |
+------------------+

ALTER TABLE (ADD COLUMN)

INTRO

The ALTER TABLE ADD COLUMN command adds a new column.

Syntax:
ALTER TABLE table_name
ADD column_name datatype;

SQL

ALTER TABLE students
ADD email VARCHAR(100);

Verify Structure

DESCRIBE students;

OUTPUT

Query OK, 0 rows affected (0.01 sec)

+--------+-------------+
| Field  | Type        |
+--------+-------------+
| id     | INT         |
| name   | VARCHAR(50) |
| grade  | CHAR(1)     |
| email  | VARCHAR(100)|
+--------+-------------+

ALTER TABLE (MODIFY COLUMN)

INTRO

The ALTER TABLE MODIFY command is your tool to change a column's data type or size.

Syntax:
ALTER TABLE table_name
MODIFY column_name new_datatype;

SQL

ALTER TABLE students
MODIFY name VARCHAR(100);

Verify Change

DESCRIBE students;

OUTPUT

Query OK, 0 rows affected (0.01 sec)

+--------+--------------+
| Field  | Type         |
+--------+--------------+

+--------+--------------+

ALTER TABLE (DROP COLUMN)

INTRO

The ALTER TABLE DROP COLUMN command removes a column you no longer need.

Syntax:

DROP COLUMN column_name;

SQL

ALTER TABLE students
DROP COLUMN email;

Verify Structure

DESCRIBE students;

OUTPUT

Query OK, 0 rows affected (0.01 sec)

+--------+--------------+
| Field  | Type         |
+--------+--------------+
| id     | INT          |
| name   | VARCHAR(100) |
| grade  | CHAR(1)      |
+--------+--------------+

DROP TABLE

INTRO

The DROP TABLE command is used to delete a table from your database.

Syntax:
DROP TABLE table_name;

SQL

DROP TABLE students;

Verify Deletion

SHOW TABLES;

OUTPUT

Query OK, 0 rows affected (0.02 sec)

+------------------+

+------------------+
| teachers         |
+------------------+

TRUNCATE TABLE

INTRO

TRUNCATE TABLE deletes all rows instantly while keeping the table structure intact.
It's faster than DELETE and resets auto-increment counters too.

Syntax:
TRUNCATE TABLE table_name;

SQL

TRUNCATE TABLE students;

Verify Table is Empty

SELECT * FROM students;

OUTPUT

Query OK, 0 rows affected (0.01 sec)

Empty set (0.00 sec)
