Insert Into Command

INTRO

              Used to insert a complete row into a table. You must match the column order.

SQL

              INSERT INTO students VALUES (1, 'Neo', 'CS', 90);

Explanation

              Adds a new student with id, name, department, and marks into the students table.

OUTPUT

              1 row inserted.

Insert Into Command (SPECIFIC Columns)

INTRO

              Inserts values only into selected columns. Useful when skipping optional fields.

SQL

              INSERT INTO students (id, name) VALUES (2, 'Trinity');

Explanation

              Only the ID and name are provided. The rest will be NULL or default.

OUTPUT

              1 row inserted.
