COUNT()

INTRO

              COUNT(column) returns the number of non-NULL values in a column.

SQL

              SELECT COUNT(marks) FROM students;

Explanation

              Counts only rows where 'marks' has a value.

OUTPUT

              +--------------+
              | COUNT(marks) |
              +--------------+
              |  4           |
              +--------------+

COUNT(*)

INTRO

              COUNT(*) returns the total number of rows, including NULLs.

SQL

              SELECT COUNT(*) FROM students;

Explanation

              Counts all rows in the table.

OUTPUT

              +-------------+
              | COUNT(*)    |
              +-------------+
              |  5          |
              +-------------+

MIN()

INTRO

              MIN() returns the smallest value from a column.

SQL

              SELECT MIN(marks) FROM students;

Explanation

              Finds the lowest marks scored.

OUTPUT

              +-------------+
              | MIN(marks)  |
              +-------------+
              |  65         |
              +-------------+

MAX()

INTRO

              MAX() returns the highest value from a column.

SQL

              SELECT MAX(marks) FROM students;

Explanation

              Finds the top score in the table.

OUTPUT

              +-------------+
              | MAX(marks)  |
              +-------------+
              |  98         |
              +-------------+

SUM()

INTRO

              SUM() adds up all the values in a column.

SQL

              SELECT SUM(marks) FROM students;

Explanation

              Gives the total of all marks entered.

OUTPUT

              +-------------+
              | SUM(marks)  |
              +-------------+
              |  423        |
              +-------------+

AVG()

INTRO

              AVG() returns the average of all numeric values.

SQL

              SELECT AVG(marks) FROM students;

Explanation

              Total of marks divided by the number of entries.

OUTPUT

              +-------------+
              | AVG(marks)  |
              +-------------+
              |  84.6       |
              +-------------+
