IF-THEN

INTRO

The IF-THEN statement executes a block of code only when a specified condition is TRUE.

Syntax:
IF condition THEN
 -- statements
END IF;

PL/SQL

DECLARE
 salary NUMBER := 12000;
BEGIN
 IF salary > 10000 THEN
    DBMS_OUTPUT.PUT_LINE('High salary');
 END IF;
END;

Explanation

Here, the salary is checked using an IF condition.
Since 12000 > 10000, the message "High salary" is printed.

OUTPUT

High salary

IF-THEN-ELSE

INTRO

The IF-THEN-ELSE control structure in PL/SQL is used to execute a block of statements 
based on a specified condition. It adds logic to our code, helping it react to different situations.

Syntax:
IF condition THEN
 -- statements
ELSE
 -- alternative statements
END IF;

PL/SQL

DECLARE
 marks NUMBER := 75;
BEGIN
 IF marks >= 50 THEN
    DBMS_OUTPUT.PUT_LINE('Passed');
 ELSE
    DBMS_OUTPUT.PUT_LINE('Failed');
 END IF;
END;

Explanation

The variable `marks` is assigned a value. The IF condition checks if it's 50 or more.
If yes, it prints "Passed". Otherwise, it prints "Failed".

OUTPUT

Passed

IF-THEN-ELSIF

INTRO

The IF-THEN-ELSIF statement allows multiple conditional checks. 
It executes the first block whose condition evaluates to TRUE.

Syntax:
IF condition1 THEN
 -- statements1
ELSIF condition2 THEN
 -- statements2
ELSE
 -- statements3
END IF;

PL/SQL

DECLARE
 marks NUMBER := 72;
BEGIN
 IF marks >= 90 THEN
    DBMS_OUTPUT.PUT_LINE('Grade A');
 ELSIF marks >= 75 THEN
    DBMS_OUTPUT.PUT_LINE('Grade B');
 ELSIF marks >= 60 THEN
    DBMS_OUTPUT.PUT_LINE('Grade C');
 ELSE
    DBMS_OUTPUT.PUT_LINE('Grade D');
 END IF;
END;

Explanation

Marks are compared in sequence. Since 72 >= 60 but < 75, 
it falls under Grade C.

OUTPUT

Grade C

BASIC LOOP

INTRO

The BASIC LOOP runs statements repeatedly until an EXIT condition is met.
It's the most flexible but needs an explicit EXIT.

Syntax:
LOOP
 -- statements
 EXIT WHEN condition;
END LOOP;

PL/SQL

DECLARE
 counter NUMBER := 1;
BEGIN
 LOOP
    DBMS_OUTPUT.PUT_LINE('Counter = ' || counter);
    counter := counter + 1;
    EXIT WHEN counter > 3;
 END LOOP;
END;

Explanation

This loop starts at 1 and runs until the counter is greater than 3.
Each iteration prints the current counter value.

OUTPUT

Counter = 1
Counter = 2
Counter = 3

WHILE LOOP

INTRO

The WHILE LOOP runs as long as the given condition is TRUE.
It's best when the number of iterations isn't known ahead of time.

Syntax:
WHILE condition LOOP
 -- statements
END LOOP;

PL/SQL

DECLARE
 i NUMBER := 1;
BEGIN
 WHILE i <= 3 LOOP
    DBMS_OUTPUT.PUT_LINE('i = ' || i);
    i := i + 1;
 END LOOP;
END;

Explanation

As long as i is less than or equal to 3, the loop continues printing the value of i.

OUTPUT

i = 1
i = 2
i = 3

FOR LOOP

INTRO

The FOR LOOP runs a fixed number of times automatically.
It handles the counter declaration and increment for you.

Syntax:
FOR counter IN [REVERSE] start..end LOOP
 -- statements
END LOOP;

PL/SQL

BEGIN
 FOR i IN 1..3 LOOP
    DBMS_OUTPUT.PUT_LINE('i = ' || i);
 END LOOP;
END;

Explanation

This loop automatically iterates from 1 to 3.
No need to manually increment the counter.

OUTPUT

i = 1
i = 2
i = 3
